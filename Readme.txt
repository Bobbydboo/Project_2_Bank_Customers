#Цель проекта "Исследование надежности заемщиков"
Повышение надежности выдачи кредитов и снижение рисков для банка

#Задачи проекта
    1. Требуется провести анализ и обработку исходных данных;
    2. Исследовать зависимость между различными параметрами клиентов и вероятностью возврата кредита;
    3. Оценить влияние признаков на вероятность возврата кредита;
    4. Построить пайплайн для предобработки данных
    5. Построить модель, предсказывающую вероятность возврата кредита для новых клиентов на основе анализа данных;
    5. Пердложить витрину данных;
    6. Подготовить отчет
